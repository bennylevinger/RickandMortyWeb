# RickandMortyWeb

How to build :
1. Use Jenkinsfile 
2. On current directory Run docker build -f WebApplication1\Dockerfile -t rickandmortapp .
Testing:
1. Run docker run -d -p 8080:80 --name myapp rickandmortapp
2. GET / Browse to http://localhost:8080/api/RandMFriends
